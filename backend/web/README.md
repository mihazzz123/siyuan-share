# Веб-фронтенд SiYuan Share

Веб-приложение на React для плагина публикации заметок SiYuan.

## Технологический стек

- **React 18**
- **TypeScript**
- **Vite**
- **React Router**
- **Axios**
- **React Markdown**

## Быстрый старт

### Установка зависимостей

```bash
npm install
```

### Режим разработки

```bash
npm run dev
```

Доступ по адресу: http://localhost:5173

### Сборка для продакшена

```bash
npm run build
```

Результаты сборки будут в директории `dist`.

### Предпросмотр сборки

```bash
npm run preview
```

## Структура проекта

```
web/
├── src/
│   ├── api/             # API запросы
│   │   ├── index.ts     # Конфигурация Axios
│   │   └── share.ts     # Запросы для публикаций
│   ├── pages/           # Компоненты страниц
│   │   ├── ShareView.tsx      # Страница просмотра публикации
│   │   ├── ShareView.css
│   │   ├── NotFound.tsx       # Страница 404
│   │   └── NotFound.css
│   ├── App.tsx          # Главный компонент приложения
│   ├── App.css
│   ├── main.tsx         # Точка входа
│   └── index.css        # Глобальные стили
├── index.html           # HTML шаблон
├── vite.config.ts       # Конфигурация Vite
├── tsconfig.json        # Конфигурация TypeScript
└── package.json
```

## Возможности

### Страница просмотра публикации

- ✅ Рендеринг Markdown контента
- ✅ Подсветка синтаксиса кода
- ✅ Проверка пароля доступа
- ✅ Отображение срока действия
- ✅ Статистика просмотров
- ✅ Адаптивный дизайн
- ✅ Поддержка темной темы

## Переменные окружения

Создайте файл `.env.local`:

```
VITE_API_URL=http://localhost:8088
```

## Разработка

### Проксирование

В режиме разработки Vite проксирует запросы по путям `/api` и `/s` на бэкенд (по умолчанию `http://localhost:8088`).
