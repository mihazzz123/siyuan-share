# Плагин публикации SiYuan (SiYuan Share)

> Быстро публикуйте выбранные документы или блоки заметок SiYuan как общедоступные веб-страницы для демонстрации, совместной работы или внешнего обмена.

该项目是 [zerx-lab/siyuan-share](https://github.com/zerx-lab/siyuan-share) 的分支版本（Fork），增加了 S3 支持、俄语本地化等功能。

## Обзор функций

- Публикация в один клик: генерация ссылки сразу после выбора документа/блока.
- Инкрементальное обновление: при повторной публикации обновляется только измененный контент.
- Обработка ресурсов: автоматическая привязка изображений, вложений и других статических файлов.
- Управление ссылками: просмотр и отзыв опубликованных ссылок в панели плагина.
- Базовая статистика просмотров (в планах).

## Настройка сервера

- Тестовый сервер: <https://share.sec-lab.cn>
  - Только для ознакомления. Стабильность и сохранность данных не гарантируются. Не храните важную информацию.
- Вы можете настроить собственный адрес сервера и параметры в панели «Настройки» плагина.

## Быстрый старт

1. Установите и включите плагин из SiYuan Bazaar.
2. Откройте документ (или выберите блок), которым хотите поделиться.
3. Используйте кнопку или пункт меню плагина для публикации.
4. При первом использовании укажите адрес сервера в настройках.
5. Получите ссылку и отправьте её получателю.
6. Для прекращения доступа удалите запись в «Списке публикаций».

## Часто задаваемые вопросы

| Вопрос | Описание |
|------|------|
| Ссылка недоступна | Проверьте доступность сервера или не был ли отзыв публикации |
| Изображения не видны | Загрузка ресурсов может быть не завершена; обновите страницу позже |
| Данные пропали на тестовом сервере | Тестовая среда периодически очищается; используйте свой сервер для стабильности |
| Конфиденциальность | Избегайте публикации личных данных; отзыв ссылки мгновенно прекращает доступ |

## Примечания и отказ от ответственности

- Плагин не несет ответственности за доступность и безопасность сторонних серверов.
- Данные на тестовом сервере могут быть удалены в любое время. Сохраняйте оригиналы заметок.
- Публикация делает контент доступным в интернете. Проверяйте заметки на наличие конфиденциальной информации перед отправкой.
