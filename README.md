# SiYuan Share Plugin (Плагин публикации для SiYuan)

> Публикуйте выбранные документы или блоки SiYuan как общедоступные веб-страницы для демонстрации, совместного просмотра или внешнего обмена.

## Возможности

- Публикация в один клик с генерацией публичной ссылки.
- Инкрементальное обновление при повторной публикации.
- Автоматическая привязка изображений и вложений.
- Управление ссылками: просмотр и отзыв в панели плагина.
- Базовая статистика посещений (планируется).

## Настройка сервера

- Тестовый сервер: <https://share.sec-lab.cn>
  - Только для тестирования. Стабильность и сохранность данных НЕ гарантируются. Не храните конфиденциальные данные.
- Вы можете настроить собственный сервер в панели настроек плагина.
- Ваш настроенный сервер: `https://share.your-domain.com`

## Быстрый старт

1. Установите и включите плагин из SiYuan Bazaar.
2. Откройте документ (или найдите блок), которым хотите поделиться.
3. Используйте пункт меню или кнопку плагина для публикации.
4. При первом использовании укажите адрес сервера (тестовый или свой собственный).
5. Скопируйте сгенерированную ссылку и отправьте её.
6. Чтобы отозвать доступ, удалите элемент в «Списке публикаций», и ссылка станет недействительной.

## FAQ (Вопросы и ответы)

| Вопрос | Ответ |
|----------|------|
| Ссылка недоступна | Проверьте доступность сервера или не была ли публикация отозвана |
| Изображения отсутствуют | Загрузка ресурсов может быть в процессе; обновите страницу позже или переопубликуйте |
| Данные пропали на тестовом сервере | Тестовая среда может периодически очищаться; используйте свой сервер для стабильности |
| Конфиденциальность | Избегайте публикации личных данных; отзыв ссылки прекращает доступ к ней |

## Примечания и отказ от ответственности

- Мы не несем ответственности за доступность, сохранность или безопасность сторонних серверов.
- Данные на тестовом сервере могут быть удалены в любое время. Всегда сохраняйте оригиналы заметок.
- Публикация делает ваш контент доступным в интернете. Предварительно проверяйте заметки на наличие конфиденциальной информации.

## Развертывание на сервере

Для автоматической сборки и обновления проекта на сервере используйте скрипт `deploy.sh`:

```bash
chmod +x deploy.sh
./deploy.sh
```

Скрипт выполнит:
1. Сборку фронтенда (React).
2. Копирование файлов в бэкенд.
3. Компиляцию Go-приложения.
4. Перезапуск системной службы `siyuan-share`.
